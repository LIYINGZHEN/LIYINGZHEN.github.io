# Commands

```bash
ls -l

# Changes the working directory to your home directory.
cd
# Changes the working directory to the previous working directory.
cd -

# i - interactive
# v - verbose
cp -iv /etc/passwd .

# i - interactive
# r - recursive
rm -ir fun
```
