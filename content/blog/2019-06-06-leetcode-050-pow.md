## Problem Statement

Implement pow(x, n), which calculates x raised to the power n (xn).

*Example 1:*

```js
Input: 2.00000, 10
Output: 1024.00000
```

*Example 2:*

```js
Input: 2.10000, 3
Output: 9.26100
```

*Example 3:*

```js
Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
```

## Solution

```go
func myPow(x float64, n int) float64 {
	if n == 0 {
		return 1
	}
	if n == 1 {
		return x
	}
	if n < 0 {
		return 1 / myPow(x, -n)
	}

	if n%2 == 0 {
		return myPow(x*x, n/2)
	} else {
		return x * myPow(x*x, n/2)
	}
}
```
